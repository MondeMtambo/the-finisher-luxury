Drop your Lottie JSON files in this folder.

Default expected path in Login.vue welcome popup:
  /animations/wave.json

Override options:
- Add ?anim=/animations/yourfile.json to the app URL
- Or set in browser console:
    localStorage.setItem('LOTTIE_ANIM', '/animations/yourfile.json')

Tip: Export JSON via Bodymovin plugin from After Effects, or grab free assets from lottiefiles.com.
Keep files small (<200KB) for fast loads.
